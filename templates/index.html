Explaining the Code

Imports

The code imports the following modules:

  • json - for working with JSON data
  • logging - for logging messages
  • tempfile - for creating temporary files
  • Modules from Django: JsonResponse, csrf_exempt, render
  • run from the robot module

Functions

home(request)

This function renders the index.html template for the home page.

execute_tests(request)

This function is responsible for executing tests based on the provided JSON payload. It performs the following steps:

  1. Parses the JSON payload from the request body
  2. Validates the payload and checks for the presence of tests
  3. Creates a temporary file with the test cases
  4. Executes the tests using the run function from the robot module
  5. Returns the test results as a JSON response

Endpoint

    http://localhost:8000/testai/tests/v1/execute

Sample POST Request

{
  "tests": [
    {
      "title": "Open google.com",
      "steps": [
        "Open Browser  https://google.com",
        "Go To  https://google.com"
      ]
    }
  ]
}
